## üèÜ Project Highlights: Advanced Routing

The core of this project is the **Advanced Routing System**. Unlike traditional websites, EduRoute utilizes 7 distinct routing patterns to provide a professional user experience:
1. **Protected Routes:** Enforce authentication before rendering components.
2. **Role Guards:** Restrict access to Instructor-only tools.
3. **Nested Routes:** Enable the sidebar-driven Course Player without page reloads.
4. **Dynamic Parameters:** Support URL-driven data fetching for thousands of lessons.
5. **Programmatic Navigation:** Redirect users based on complex business logic (e.g., role-based login).
6. **Lazy Loading:** Ensure lightning-fast initial page loads through code splitting.
7. **Error Handling:** Robust 404 and 403 pages for optimized error UX.
